Acerca de

La Consola de Administración para Docker es una aplicación web diseñada para facilitar la gestión de contenedores Docker.

Características Principales

  • Administración de Imágenes: Explora, elimina y carga o envia nuevas imágenes Docker desde el repositorio oficial y repositorios personalizados.
  • Gestión de Contenedores: Permite explorar, crear, iniciar, detener y eliminar contenedores de Docker de forma intuitiva.

Esta aplicación ha sido desarrollada con fines educativos y de aprendizaje en la administración de contenedores docker. Si tienes alguna pregunta o comentario, no dudes en contactarme en ing.bernardosegura@gmail.com.
